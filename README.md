# Off The Lamp

### 프로젝트 기간
2023.08.24 ~ 2023.09.27

### 개발 인원
6명(팀프로젝트)

### 개발 의도
거리두기로 인해 문화산업 침체에도 불구하고, 자동차극장은 특유의 프라이빗함으로
성수기를 맞이했으나 관련 정보 한번에 모아 놓은 사이트가 없고, 예매방식도 과거에 머물러
있는 문제를 해결해 보기 위해 개발을 하였습니다.

### 프로젝트의 목적

### 개발 환경
JSPMVC2 Model을 활용하여 개발
